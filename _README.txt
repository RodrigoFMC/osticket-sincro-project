O projeto apresentado neste relatório descreve o desenvolvimento de um plugin para o osTicket. O plugin tem como principal funcionalidade a estruturação dos dados dos tickets para o reporte de avarias nos radares/cabines do projeto Sistema Nacional de Controlo de Velocidade (SINCRO), gerido pela ANSR.

O presente projeto tem como finalidade melhorar a eficiência no tratamento e acompanhamento das avarias reportadas, garantindo uma gestão mais eficaz e organizada das solicitações de suporte relacionadas com o sistema SINCRO.

Todo o código fonte e a sua documentação, assim como a documentação de como instalar o plugin no osTicket, podem ser encontrados neste repositório na pasta 03_Implementacao.
